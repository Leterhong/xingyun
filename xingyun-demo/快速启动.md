# 快速启动指南

## 🚀 5分钟快速上手

### 步骤1：安装依赖

```bash
npm install
```

### 步骤2：启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动。

### 步骤3：配置参数

首次访问会自动弹出配置对话框，需要填写以下必填项：

#### 3.1 获取魔珐星云配置

1. 访问 [魔珐星云平台](https://xingyun3d.com/)
2. 注册/登录账号
3. 进入"应用中心"
4. 点击"创建驱动应用"
5. 选择数字人角色、音色、表演风格
6. 复制 `App ID` 和 `App Secret`

#### 3.2 获取大模型API Key

**推荐：使用豆包（字节跳动）**

1. 访问 [火山引擎](https://console.volcengine.com/ark)
2. 注册/登录账号
3. 创建推理接入点
4. 选择模型：`doubao-pro-32k`
5. 复制生成的 `API Key`

**其他选择：**
- [DeepSeek](https://platform.deepseek.com/) - 推理能力强，价格便宜
- [通义千问](https://dashscope.console.aliyun.com/) - 中文能力优秀
- [OpenAI](https://platform.openai.com/) - 需要特殊网络环境

#### 3.3 填写配置

在配置对话框中填写：

```
魔珐星云配置：
- App ID: [粘贴你的App ID]
- App Secret: [粘贴你的App Secret]

大模型配置：
- 模型: doubao-pro-32k
- API Key: [粘贴你的API Key]

语音识别配置：（可选，暂时跳过）
```

点击"保存配置"，然后刷新页面。

### 步骤4：开始使用

配置完成后，可以：

1. **文字对话**
   - 在右侧聊天面板输入问题
   - 按 `Ctrl + Enter` 发送
   - 数字人会用语音和动作回答

2. **快捷操作**
   - 点击"自我介绍"体验功能
   - 尝试其他快捷按钮

3. **打断说话**
   - 数字人说话时可随时打断
   - 点击"打断说话"按钮

## ⚠️ 常见启动问题

### 问题1：页面白屏

**原因**：依赖未安装完整

**解决**：
```bash
# 清除依赖重新安装
rm -rf node_modules package-lock.json
npm install
```

### 问题2：SDK加载失败

**原因**：网络问题或CDN访问受限

**解决**：
1. 检查网络连接
2. 尝试使用VPN
3. 等待片刻后刷新页面

### 问题3：数字人不显示

**原因**：配置错误或SDK未初始化

**解决**：
1. 检查配置是否正确填写
2. 查看浏览器控制台错误信息
3. 确认使用 `localhost` 而非IP地址访问
4. 刷新页面重新连接

### 问题4：通过IP访问报错

**原因**：SDK不支持IP地址访问

**解决**：
使用 `http://localhost:5173` 而不是 `http://192.168.x.x:5173`

### 问题5：数字人无声音

**原因**：浏览器音频未启用

**解决**：
1. 检查浏览器静音设置
2. 检查系统音量
3. 刷新页面重新初始化

## 💡 开发技巧

### 查看调试信息

点击数字人面板下方的"显示调试信息"按钮，可以查看：
- 帧率（FPS）
- 网络延迟
- 渲染状态

### 查看网络信息

数字人面板会实时显示：
- 连接状态
- 网络延迟（RTT）
- 下载速率

### 清除配置

如需重新配置，在浏览器控制台执行：
```javascript
localStorage.clear()
```
然后刷新页面。
