## 🎨 设计语言

### 色彩系统
```css
/* 主色调 */
Primary: #409eff (蓝色)
Success: #67c23a (绿色)
Warning: #e6a23c (橙色)
Danger: #f56c6c (红色)

/* 背景色 */
主背景: linear-gradient(135deg, #667eea, #764ba2)
卡片背景: rgba(255, 255, 255, 0.98)
次要背景: #fafbfc

/* 文字色 */
主文字: #303133
常规文字: #606266
次要文字: #909399
```

### 圆角系统
```css
小圆角: 8px  (按钮、标签)
中圆角: 12px (卡片、输入框)
大圆角: 16px (主容器)
胶囊: 20px  (状态指示器)
```

### 阴影系统
```css
轻阴影: 0 2px 8px rgba(0,0,0,0.05)
中阴影: 0 4px 12px rgba(0,0,0,0.1)
重阴影: 0 8px 32px rgba(0,0,0,0.12)
悬停阴影: 0 12px 40px rgba(0,0,0,0.15)
```

### 动画时长
```css
快速: 0.2s (按钮、hover)
标准: 0.3s (卡片、模态框)
缓慢: 0.4s (页面、大型动画)
```

## 🎬 动画效果

### 1. 页面加载动画
```css
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
```

### 2. 消息进入动画
```css
@keyframes messageIn {
  from { opacity: 0; transform: translateY(20px) scale(0.95); }
  to { opacity: 1; transform: translateY(0) scale(1); }
}
```

### 3. 脉动动画
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.6; }
}
```

### 4. 悬停效果
- 卡片：上浮2px + 阴影增强
- 按钮：上浮2px + 阴影增强
- 图标：缩放1.1倍

## 🔧 使用的CSS技术

### 1. CSS变量
```css
:root {
  --primary-color: #409eff;
  --border-color: #dcdfe6;
}

.element {
  color: var(--primary-color);
}
```

### 2. 玻璃态效果
```css
background: rgba(255, 255, 255, 0.98);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.8);
```

### 3. 渐变背景
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 4. 文字渐变
```css
background: linear-gradient(135deg, #667eea, #764ba2);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

### 5. 深层选择器
```css
.input-area :deep(.el-textarea__inner) {
  /* Element Plus 组件内部样式 */
}
```

## 📊 性能优化

### 1. CSS优化
- ✅ 使用 `will-change` 提示浏览器优化
- ✅ 动画使用 `transform` 而非 `top/left`
- ✅ 减少重绘和回流
- ✅ 使用 GPU 加速（`transform3d`）

### 2. 加载优化
- ✅ 关键CSS内联

- ✅ 字体预加载

### 3. 渲染优化
- ✅ 使用 `contain` 属性

## 🎯 输入框特别优化 ⭐

针对您提出的"让文本输入框能展示出来"的需求，做了以下重点优化：

### 视觉突出
1. **边框加粗**：从1px增加到2px
2. **圆角增大**：从8px增加到12px
3. **阴影增强**：添加默认阴影，聚焦时发光效果
4. **背景渐变**：输入区域整体使用渐变背景

### 交互反馈
1. **悬停效果**：边框颜色变化
2. **聚焦效果**：蓝色发光+阴影扩散
3. **按钮动画**：悬停时上浮+阴影增强

### 布局优化
1. **增加内边距**：12px → 更舒适的输入体验
2. **区域独立**：渐变背景区分
3. **提示图标**：添加💡图标，更友好

### 效果展示

### 完全支持
- ✅ Chrome 90+
- ✅ Edge 90+
- ✅ Firefox 88+
- ✅ Safari 14.1+

### 部分支持

### 降级方案
- backdrop-filter 不支持时使用纯色背景
- CSS变量不支持时使用回退值
- Grid布局不支持时使用Flexbox
